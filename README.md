## Website Pengaduan Masyarakat

Website Pengaduan Masyarakat adalah sebuah platfrom yang digunakan untuk memfasilitasi dalam melaporkan masalah atau keluhan yang terjadi di sekitar. Dengan adanya platfrom ini, masyarakat dapat dengan mudah melaporkan masalah tersebut secara online dan pihak yang berwenang segera menindaklanjuti.

## Fitur

-   Masyarakat dapat melaporkan masalah atau keluhan
-   Pihak yang berwenang dapat memantau dan menindaklanjuti laporan tersebut
-   Masyarakat dapat meilhat status laporan yang telah mereka buat

## Teknologi yang digunakan

-   Laravel sebagai framework PHP
-   MySQL sebagai database
-   Bootstrap sebagai framework CSS

## Cara Menggunakan

-   Jalankan perintah 'php artisan migrate' untuk membuat table yang dibutuhkan
-   Jalankan perintah 'php artisan serve' untuk menjalankan aplikasi
